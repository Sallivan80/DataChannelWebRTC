WebRTC

Un progetto aperto libero che permette una comunicazione in Real-Time su browser web tramite semplici API JavaScript.
I componenti WebRTC sono stati ottimizzati per servire al meglio questo scopo

MQ Telemetry Transport.

Un protocollo Machine-To-Machine, progettato come un trasporto estremamente leggero.
Utile per i collegamenti con località remote, disegnato per le situazioni in cui è richiesto un basso impatto e dove la banda è limitata.

WebRTC Demo

Questa demo fa uso delle API WebRTC per instaurare una chat di comunicazione molti a molti Peer-To-Peer.
Il Signaling Server è scritto in Java, e sfrutta le WebSocket per permettere lo scambio di metadati iniziale, indispensabili per instaurare una comunicazione.
